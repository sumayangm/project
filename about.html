About the Pokémon Go Webstore (Redesigned)

This website is a redesign project based on the official Pokémon Go Web Store. The goal is to present a cleaner and responsive version using only HTML, CSS, and JavaScript. The original site allows users to buy PokéCoins and bundles for use in the Pokémon Go mobile game.

This project was created as part of an academic requirement to practice web design skills. No sales are made through this website. All buttons and features are simulated for demonstration purposes only.
